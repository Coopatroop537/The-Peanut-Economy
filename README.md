# The Peanut Economy

The Peanut Economy is a Balatro-inspired slot machine game where players gamble with peanuts and buy randomized cards from a shop.

## Game Features
- **Slot Machine Gameplay**: Players spin the slot machine to win peanuts
- **Card Shop System**: Buy randomized modifier cards to enhance gameplay
- **Balatro Inspiration**: Dynamic card effects that change how the game plays
- **Peanut Currency**: Earn and spend peanuts throughout your gaming session

## Project Structure
- Assets/
  - Scripts/
    - Core/ (PeanutManager, GameManager)
    - Gameplay/ (SlotMachine, CardSystem)
    - Shop/ (ShopManager, CardPool)
    - UI/ (UIManager, displays)
  - Scenes/
  - Prefabs/
  - Resources/

## Getting Started
Clone the repository and open it in Unity 2022 LTS or newer.
